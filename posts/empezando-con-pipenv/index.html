Empezando con Pipenv

lecovi

2018-01-31 12:26

Código fuente

En este brevísimo artículo te explicamos cómo empezar a usar pipenv.

Instalado Pipenv

pipenv es una herramienta para manejar paquetes y entornos virtuales de manera sencilla en tu entorno de desarrollo. Te recomiendo que le eches un ojo a la Documentacion Oficila. Ahora te voy a mostrar cómo instalar pipenv y cómo usarlo en tus proyectos.

Instalar Pipenv

La forma más fácil de instalar pipenv es a través de pipsi. Si no tenés pipsi instalado, lo podés instalar con:

$ curl https://raw.githubusercontent.com/mitsuhiko/pipsi/master/get-pipsi.py | python3

Para poder ejecutar desde la línea de comandos tenés que agregar a tu variable de entorno PATH la ruta donde está el ejecutable de pipsi.

Con bash:

export PATH=$PATH:~/.local/bin/pipsi

Con fish:

set PATH $PATH ~/.local/bin/pipsi

Y después tenés que instalar pew y pipenv:

$ pipsi install pew
$ pipsi install pipenv

Comandos útiles

  • pipenv --three install <paquete>: instala en el virtualenv.
  • pipenv --three install -r <archivo_de_requerimientos>: instala los requerimientos en el virtualenv.
  • pipenv shell: activa el virtualenv en la shell.
  • pipenv run <archivo>: ejecuta desde el virtualenv.
  • pipenv --rm: borra el virtualenv.
  • pipenv --update: actualiza pip & pipenv.
  • pipenv --venv: muestra el PATH del virtualenv.
